# JobJourney - Frontend (Client)

This is the React-based frontend for JobJourney, a career tracking platform.

## 🚀 Key Libraries
- **React 19**: Modern UI development.
- **Tailwind CSS 4**: Modern styling with `@theme` blocks.
- **TanStack Query**: Sophisticated server-state management.
- **Framer Motion**: Premium interactive animations.
- **Recharts**: Data-driven career analytics visualization.
- **@hello-pangea/dnd**: Kanban board implementation.

## 📁 Folder Structure
- `src/components`: UI components (Cards, Forms, Layouts).
- `src/pages`: Main application views.
- `src/hooks`: Custom React hooks for business logic.
- `src/context`: Authentication and global state.
- `src/utils`: API services and common helpers.

## 🛠️ Development
```bash
npm install
npm run dev
```
The client communicates with the backend via the API base URL defined in the environment or configuration.
